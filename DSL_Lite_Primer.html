DSL Lite

Cybersecurity Lakehouse Accelerator  ·  Medallion Architecture  ·  OCSF 1.7.0
Databricks Forward Deployed Engineering
Built for the Cyber Lakehouse
DSL Lite is a declarative, configuration-driven framework that transforms raw security logs from any source into OCSF-compliant, analytics-ready tables on Databricks — with no custom Spark code required. Security teams get a production-grade Bronze → Silver → Gold medallion pipeline in hours, not weeks or months.
Medallion Architecture
Bronze — Ingest & Amend Auto Loader incremental ingestion. Preserves raw payload (VARIANT or STRING) + 9 standard metadata columns + dsl_id.
›
Silver — Parse & Curate Extracts typed fields via JSON flattening, regex, or from_csv. Applies filters and business logic.
›
Gold — Map & Normalize Maps to OCSF-compliant schema. Shared Delta Sinks support multi-source aggregation into unified event tables.
Key Capabilities
  • Zero Spark code — pipelines defined entirely in YAML preset files
  • OCSF 1.7.0 compliance — gold tables map to standard event classes with metadata STRUCT versioning
  • Multi-format ingestion — JSON / JSON Lines, CSV (with or without headers), Syslog / text
  • Streaming & batch — continuous streaming or availableNow triggered runs
  • Schema enforcement — OCSF DDL notebook provisions gold tables with typed, versioned schemas for downstream query performance and data governance
  • Lookup enrichment — join bronze/silver fields against Delta or CSV reference tables
  • Secret management — Databricks Secrets referenced inline in preset YAML
  • Liquid Clustering — all layers cluster by time for fast security query performance
  • Delta Column Mapping — rename OCSF fields post-deployment with no data rewrite
  • One-command deployment — Declarative Automation Bundles (DABs) per log source
  • Agentic development — Agent skills augment AI coding tools and Databricks Genie for preset authoring, validation, and pipeline Q&A without leaving the workspace
Execution Modes
Spark Declarative Pipeline SDP
Lakeflow-managed pipeline. Serverless, continuous or triggered. Gold tables as Delta Sinks for cross-vendor aggregation.
Spark Structured Streaming SSS
Classic compute job. Three independent tasks (bronze / silver / gold) or a single medallion job. Full control over cluster configuration.
Streaming ingest (on request): Native file-based ingestion via Auto Loader is the default. Streaming source support — Kafka, Zerobus, Azure Event Hubs, AWS Kinesis — is available as a development feature and can be enabled per engagement. Contact your Databricks team to scope.

Developer Experience
🤖
Preset Agent
Foundation-model-powered notebook. Introspects a Unity Catalog table and generates a first-draft preset.yaml — entirely inside the workspace. No data leaves the customer environment.
🔍
Preset Explorer
Interactive notebook for iterating on presets in batch mode against real sample data. Preview bronze, silver, and gold outputs before any pipeline deployment.
📊
Pipeline Profiler
Pre-deployment QA notebook. Runs schema diff, null-rate comparison, E2E sample run, and OCSF coverage checks against legacy tables. Saves a timestamped Markdown report to a Volume.
Agentic & self-service development: Step-by-step tutorials guide preset authoring end-to-end. Agent skills packaged in .agents/skills/ augment AI coding tools (Claude Code, Cursor, Copilot) and Databricks Genie — enabling natural-language preset Q&A, pipeline troubleshooting, and OCSF mapping assistance directly inside the workspace.
